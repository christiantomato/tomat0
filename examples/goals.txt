IDEAS FOR TOMAT0 


1. Keyword: sout

Used for printing to the terminal.
Example:

sout(myVariable)

2. Right Chevron Comments

Example:

> this is a comment
> print out the value 
sout(value)

3. No Delimeters

No multiplie line statements are allowed, therefore no delimeters have to be used

Example:

int x = 5
string y = "hello"
sout(x)
sout(y)
